TDD의 GREEN 단계를 실행하세요:

1. RED 단계에서 실패하는 테스트가 있는지 확인
2. 테스트를 통과시키는 데 필요한 최소한의 코드 작성:
   - 작동할 수 있는 가장 단순한 솔루션 사용
   - 추가 기능을 넣지 마세요
   - 이 단계에서는 값을 하드코딩해도 괜찮습니다
   - 완벽한 설계가 아닌 테스트 통과에 집중
3. 모든 테스트를 실행하여 확인:
   - 새 테스트가 이제 통과하는지
   - 기존 테스트가 여전히 모두 통과하는지
4. 모든 테스트가 통과하면 REFACTOR 단계로 진행할 준비 완료

기억하세요:
- 테스트를 통과시키는 데 충분한 코드만 작성 - 그 이상은 안 됩니다
- 아직 중복에 대해 걱정하지 마세요
- 아직 테스트되지 않은 기능을 추가하지 마세요
